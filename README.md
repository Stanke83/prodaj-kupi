# ProdajKupi 🛒

Moderna besplatna platforma za oglase — klon KupujemProdajem.

## 📁 Stranice

| Fajl | Opis |
|------|------|
| `index.html` | Početna — hero, najnoviji oglasi |
| `pretraga.html` | Pretraga sa filterima, sortiranjem i paginacijom |
| `postavi-oglas.html` | Forma za postavljanje oglasa (6 koraka) |
| `uredi-oglas.html` | Uređivanje postojećeg oglasa |
| `profil.html` | Profil korisnika, moji oglasi, sačuvani, podešavanja |

## ✨ Funkcionalnosti

- ✅ Registracija i prijava korisnika
- ✅ Postavljanje oglasa (foto, kategorija, stanje, cena, lokacija, kontakt)
- ✅ Pretraga sa filterima (kategorija, grad, stanje, cena od-do)
- ✅ Sortiranje (najnoviji, cena, najstariji)
- ✅ Paginacija (16 oglasa po stranici)
- ✅ Profil korisnika sa statistikama
- ✅ Uređivanje i brisanje oglasa
- ✅ Sačuvani/favoriti oglasi
- ✅ Promjena lozinke i podataka profila
- ✅ Foto galerija u detaljnom prikazu
- ✅ Responzivan dizajn (mobile-first)

## 🛠 Tehnologije

- HTML5, CSS3, Vanilla JavaScript
- localStorage (baza podataka)
- Google Fonts (Inter)

## 🚀 Pokretanje

Otvori `index.html` u browseru — nema potrebe za serverom!
